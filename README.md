# Currency Converter 

A responsive currency converter built using HTML, CSS, and JavaScript.

## Features
- Real-time currency conversion using Exchange Rate API
- Dynamic country flag updates
- Interactive UI with DOM manipulation

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)
- Fetch API

## Learning Outcome
This was my first project using JavaScript and APIs, where I learned DOM manipulation, asynchronous programming, and API integration.
